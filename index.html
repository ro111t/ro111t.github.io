How to Apply CSS to HTML

To link an external CSS file to an HTML file, you can use the following function: 

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link external Javascript files to HTML documents, you can use the following syntax: 

<script src="script.js"></script>
Along with that, JavaScript code can be written directly in the HTML file within the <script> tags: 
<script>console.log('Hello, World!')</script>
